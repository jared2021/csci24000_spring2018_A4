The input of this program is textfile that we want to copy and the button prompts they will use to run the program
The output of this program is printing out the contents of the textfile we copied
When the user hits 1 the program will scan the textfile, parse the data and store it in the Student and Address class respectively, and store the Student class in our Stack ADT
When the user hits 2 the program will pop out each Student and print them out
When the user hits 3 the program will end
